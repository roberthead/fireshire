# Story: Create a Thing

## Summary

AS a logged-in user
ON the thingamabob page
I WANT to create a new thing
SO THAT I can enjoy the thing

## Acceptance Criteria

- User can click "New Thing"
- A name must be provided (required)
- Optional: description, image
- After creation, user is redirected to the thingamabob page

## Notes

- Default thing name is 'thing'
- Thing slug must be URL-safe
